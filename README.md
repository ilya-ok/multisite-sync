# Multisite Price Sync

Плагин для синхронизации цен WooCommerce товаров между сайтами WordPress Multisite по SKU.

## Описание

Этот плагин автоматически синхронизирует цены товаров на всех сайтах мультисайта WordPress. Когда вы обновляете цену товара на одном сайте, она автоматически обновляется на всех остальных сайтах, где есть товар с таким же SKU.

## Требования

- WordPress Multisite
- WooCommerce 5.0+
- PHP 7.4+

## Возможности

### ✅ Автоматическая синхронизация
- Автоматическое обновление цен при сохранении товара
- Синхронизация обычной цены (Regular Price)
- Синхронизация цены со скидкой (Sale Price)
- Поддержка вариаций товаров

### ✅ Массовое редактирование
- Импорт цен из CSV файла
- Ручное редактирование таблицей
- Автопоиск товаров по SKU
- Автозаполнение текущих цен

### ✅ Гибкие настройки
- Включение/отключение автоматической синхронизации
- Выбор, какие цены синхронизировать
- Режим отладки
- Информация о сайтах мультисайта

## Установка

1. Загрузите папку `multisite-price-sync` в `/wp-content/plugins/`
2. Активируйте плагин для сети (Network Activate)
3. Перейдите в Network Admin → Цены Multisite

## Использование

### Автоматическая синхронизация

1. Откройте товар на любом сайте мультисайта
2. Измените цену
3. Сохраните товар
4. Цена автоматически обновится на всех сайтах с товарами с таким же SKU

### Массовое редактирование через CSV

1. Перейдите: **Network Admin → Цены Multisite**
2. Создайте CSV файл в формате:
   ```
   SKU,Regular Price,Sale Price
   ABC123,1000,800
   DEF456,2000,1500
   GHI789,500,
   ```
3. Загрузите файл через форму
4. Нажмите "Импортировать и синхронизировать"

### Ручное редактирование

1. Перейдите: **Network Admin → Цены Multisite**
2. В разделе "Ручное редактирование":
   - Введите SKU товара
   - Товар будет автоматически найден
   - Текущие цены подставятся автоматически
   - Измените цены
   - Нажмите "Добавить товар" для добавления строк
3. Нажмите "Обновить и синхронизировать"

### Настройки

Перейдите: **Network Admin → Цены Multisite → Настройки**

**Доступные настройки:**
- **Автоматическая синхронизация** - вкл/выкл автосинхронизации при сохранении
- **Синхронизировать обычную цену** - включить синхронизацию Regular Price
- **Синхронизировать цену со скидкой** - включить синхронизацию Sale Price
- **Режим отладки** - логирование в debug.log

## Как это работает

### Поиск товаров

Плагин использует **SKU (артикул)** для идентификации товаров между сайтами. Товары с одинаковым SKU считаются одним и тем же товаром.

### Процесс синхронизации

1. Обновляется цена товара на сайте A
2. Плагин получает SKU товара
3. Плагин переключается на сайт B
4. Ищет товар с таким же SKU
5. Обновляет цены товара
6. Повторяет для сайтов C, D, E...

### Предотвращение рекурсии

Плагин использует флаг `$is_syncing` для предотвращения бесконечных циклов обновления.

## FAQ

### Что будет, если товар не найден на другом сайте?

Плагин просто пропустит этот сайт и продолжит синхронизацию на остальных.

### Нужно ли указывать SKU для всех товаров?

Да, товары без SKU не будут синхронизироваться.

### Синхронизируются ли вариации товаров?

Да, если у вариации есть свой SKU.

### Можно ли отключить автосинхронизацию?

Да, в настройках плагина.

### Можно ли синхронизировать только обычную цену без скидки?

Да, в настройках можно выбрать, какие цены синхронизировать.

## Техническая информация

### Хуки WordPress

**Используемые хуки:**
- `woocommerce_update_product` - отслеживание обновления товара
- `woocommerce_new_product` - отслеживание создания товара
- `updated_post_meta` - отслеживание изменения мета-данных

### Multisite функции

**Используемые функции:**
- `get_sites()` - получение списка сайтов
- `switch_to_blog()` - переключение на другой сайт
- `restore_current_blog()` - возврат на исходный сайт

### WooCommerce функции

**Используемые функции:**
- `wc_get_product_id_by_sku()` - поиск товара по SKU
- `wc_get_product()` - получение объекта товара
- `$product->get_sku()` - получение SKU
- `$product->get_regular_price()` - получение обычной цены
- `$product->get_sale_price()` - получение цены со скидкой
- `$product->set_regular_price()` - установка обычной цены
- `$product->set_sale_price()` - установка цены со скидкой
- `$product->save()` - сохранение товара
