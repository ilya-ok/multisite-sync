## Выгрузка товаров в XML
Инструмент для выгрузки товаров WooCommerce в XML-файл.  
Прототип есть в `wp-content/plugins/woocommerce multisite product sync/admin/partials/plugin-name-admin-products-download.php` — выгружает id, название, цену, описание и атрибуты товара.

**Что нужно:**
- Выбор категории для выгрузки
- Настройка полей (какие данные включать)
- Скачивание готового XML

## Замена топонима в описаниях товаров по городам
Инструмент для мультисайта: автоматически заменяет название города в описаниях товаров при переключении между сайтами (например, «в Екатеринбурге» → «в Челябинске»).  
Прототип есть в `wp-content/plugins/woocommerce multisite product sync/admin/partials/plugin-name-admin-product-descr.php`.

**Что нужно:**
- Настройка топонима для каждого сайта мультисайта
- Массовая замена по всем товарам
- Предпросмотр результата перед сохранением
